## 分析音频

以1s为节拍界限,当发现下个节拍比上个节拍一直高的时候,就一直找,知道找到没有上个节拍高的地方, 若是有起伏,则根据(1s内) 高低高,低高高, 高高低,低高低,节拍这个东西,

### 方式2

1. 要么是2个,要么是4个
2. 节拍的分析范围在1秒内. 当两个接近一秒时,则选择两个,当4个接近一秒时,则选择4个.
3. 若一直高,那么就一直向后分析,直到找到低的,然后落下.
4. 若一直低.那么就一直向后分析,直到找到高的,然后升起.
5. 若2个,或者4个
  * 若第一个为高
    * 若最后一个为低,则前倾
    * 若最后一个为高,若是2拍,则分成2个动作
  * 若第一个为低


### 方式3

直接找到变化比较大的节拍,在进行动作判断.

当找到动作变化比较大的节拍时,还得分析时间. 若时间太短,则继续向后分析. 若时间已经比较长,则直接开始运动.
